# EduVision AI
A Python-powered website that uses ResNet50 and K-Means to analyze color-based accessibility of educational visual aids for CVD conditions. Made for a group research entitled "Development of a Web-Based AI Color Classifier to Measure the Accessibility of Educational Visual Aids Under Simulated Color Vision Deficiency (CVD) Conditions."

## Directions for Use
1. Download Conda (optional, used to properly run repositories)
2. Download required repositories (Numpy, Pillow, Tensorflow, ScikitLearn, ScikitImage)
3. Create folder "content" within website
4. Download model weights from [this Google Drive](https://drive.google.com/drive/folders/1cU0Q-3xHQNkrhzEqEQRkAO_m_dmwL6nD?usp=drive_link)
5. Activate Conda environment
6. Run app.py
